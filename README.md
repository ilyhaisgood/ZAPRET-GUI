# ZAPRET-GUI
Приложение на основе Electron, сделанное под запрет - flowseal, упрощающая легкость работы. Сделано с помощью Нейросети!

## Быстрый старт

1. Скачайте и распакуйте последнюю версию приложения из раздела [**Releases**](https://github.com/ilyhaisgood/ZAPRET-GUI/releases)
2. Скачайте актуальную версию `zapret` с официального репозитория: https://github.com/Flowseal/zapret-discord-youtube/releases
3. Из архива Flowseal перенесите все файлы в папку `zapret` в `resources/zapret`.
4. Запустите `ZapretGUI.exe`.
5. В настройках выберите нужный профиль.
6. На главной странице нажмите кнопку **Пуск**.

После этого обход будет активирован.

## Распрастранённые проблемы и их решения

### После запуска приложения, которое было установлено через exe, и после выбора профиля не запускается winws
- Перейдите в папку, где установлено приложение.
- Откройте каталог /resource/zapret.
- Удалите всё содержимое этой папки.
- Скачайте последнюю версию [Zapret](https://github.com/Flowseal/zapret-discord-youtube/releases) от репозитории Flowseal.
- Распакуйте архив.
- Скопируйте всё содержимое распакованного архива в папку /resource/zapret, заменив существующие файлы при необходимости.
