# ZAPRET-GUI
Приложение на основе Electron, сделанное под запрет - flowseal, упрощающая легкость работы. Сделано с помощью Нейросети!

# Принцип работы
1. Скачиваете архив / exe файл , разархивируете архив / выполняете загрузку exe файла в любое удобное место
2. скачать с офф релиза https://github.com/Flowseal/zapret-discord-youtube/releases последнюю версию обхода, перенести папку с запретом в ПАПКА/resources/zapret(сюда уже .bat и все остальные файлы кидать как надо)
3. Запустить приложение ZapretGUI.exe
4. Выбрать профиль(версию bat) в настройках
5. Вернуться на главную и нажать на кнопку
6. Все, обход запущен
